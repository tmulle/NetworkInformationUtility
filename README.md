# NetworkInformationUtility

Class that prints out all the information about your Network Interfaces

### Compile: 
```mvn clean package```

## Run

Maven Exec: ```mvn exec:java```

or

Command Line: ```java -jar target/networkinterface_info-1.0-fat.jar```
